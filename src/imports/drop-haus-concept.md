1. Opening Experience (Pre-Hero)

When the user lands on the site:

The screen is black.

A small glowing line appears in Drop Green #DBEC62.

It pulses like a loading signal.

Text fades in:

After 1.5 seconds:

Logo fades in.

Then the homepage begins.

This small moment creates anticipation immediately.

3. Scroll Trigger — The Drop Effect

When the user scrolls:

The hero text breaks apart into floating letters.

They move into the background.

The camera effect zooms into the next section.

This creates a cinematic transition.

4. The Problem Section

Dark minimal section.
